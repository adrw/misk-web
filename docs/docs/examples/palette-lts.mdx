# Tab: Palette Long Term Support (LTS)

The `palette-lts` tab is an example tab showcasing elements and functionality related to forms, network, and shared styling.

It uses older deprecated `@misk/simpleredux` APIs to test long term support of older tabs.

[See the code.](https://github.com/cashapp/misk-web/tree/master/examples/tabs/palette-lts)
