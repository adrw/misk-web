# Getting Started

Here's how to get started building a new Misk-Web tab.

```
npm install -g @misk/cli
miskweb new
./new-tab.sh
cd my-new-tab
miskweb ci-build
miskweb start
```

Note: the `miskweb new` command only downloads the `new-tab.sh` script, it will need to be run manually.
